Project Overview

This project involved designing and implementing a complete electrical and embedded control system for coffee roasting machines with capacities ranging from 1 kg to 25 kg. The focus was on reliability, repeatability, and user-friendly operation.

Responsibilities

  • Electrical system design and wiring
  • Embedded firmware development
  • Desktop control application development
  • Flutter-based Android control application
  • Machine commissioning and customer training

Technologies Used

Embedded C Flutter PCB Design (Eagle) Industrial Sensors Motor Control
